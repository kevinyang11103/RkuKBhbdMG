# 仓库管理系统 java1166

## 项目概述

仓库管理系统是一款基于Spring Boot和Vue的前后端分离的计算机项目程序设计管理系统。该系统集成了商城网站的基本功能，使用Java作为主要开发语言，采用MVC架构和MyBatis作为数据访问层，支持MySQL数据库，致力于提供高效便捷的仓库管理解决方案。

## 技术栈

- **后端：**
  - Java
  - Spring
  - Spring Boot
  - MVC
  - MyBatis
  - MySQL
  - Maven

- **前端：**
  - HTML
  - JavaScript
  - CSS

- **开发工具：**
  - IntelliJ IDEA / Eclipse
  - Visual Studio Code

## 功能模块

系统包含但不限于以下功能模块：

- 用户管理
- 商品管理
- 仓库管理
- 订单管理
- 报表统计

## 系统角色

- 系统管理员
- 仓库管理员
- 销售人员
- 客户

## 运行环境

- JDK 1.8 或更高版本
- MySQL 数据库
- Maven 项目管理工具

## 部署步骤

1. 环境准备：确保JDK、MySQL和Maven已正确安装。
2. 数据库配置：根据提供的脚本创建数据库和表结构。
3. 后端部署：使用IDEA或Eclipse导入项目，编译并运行。
4. 前端部署：使用Visual Studio Code打开前端项目，编译并运行。
